1. Project is a collection of applications along with their settings.

2. Project can have at least one application and at most n number of applications.

3. Application is a specify task of the project.

4. Each application has its own purpose.

5. When we install Django then django-admin fill will be installed is scripts folder, this file is used to create Django project.

6. When we create Django project then manage.py file will be created in the project folder, this file is used to create an application.
